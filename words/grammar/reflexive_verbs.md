# -ομαι / -αμαι / -ιεμαι
## возвратные и страдательные глаголы (парадигма)
### -ome / -ame / -iéme
Греческие возвратные/медиопассивные глаголы имеют три группы окончаний:

**Группа -ομαι (напр. γίνομαι, έρχομαι, σκέφτομαι):**
| Лицо | Форма |
|------|-------|
| εγώ | -ομαι |
| εσύ | -εσαι |
| αυτός/ή/ό | -εται |
| εμείς | -όμαστε |
| εσείς | -εστε / -όσαστε |
| αυτοί/ές/ά | -ονται |

**Группа -αμαι (напр. κοιμάμαι, λυπάμαι):**
| Лицо | Форма |
|------|-------|
| εγώ | -αμαι |
| εσύ | -άσαι |
| αυτός/ή/ό | -άται |
| εμείς | -όμαστε |
| εσείς | -όσαστε |
| αυτοί/ές/ά | -ούνται |

**Группа -ιεμαι (напр. αγαπιέμαι, συναντιέμαι):**
| Лицо | Форма |
|------|-------|
| εγώ | -ιέμαι |
| εσύ | -ιέσαι |
| αυτός/ή/ό | -ιέται |
| εμείς | -ιόμαστε |
| εσείς | -ιόσαστε |
| αυτοί/ές/ά | -ιούνται |

• Εγώ κοιμάμαι, εσύ κοιμάσαι, αυτός κοιμάται, εμείς κοιμόμαστε, εσείς κοιμόσαστε, αυτοί κοιμούνται.
• Εγώ συναντιέμαι, εσύ συναντιέσαι, αυτός συναντιέται, εμείς συναντιόμαστε, εσείς συναντιόσαστε, αυτοί συναντιούνται.
