# μου
## мой, моя, моё, мои (притяжательное местоимение)
### mu
Притяжательные местоимения ставятся ПОСЛЕ существительного с артиклем.
| Лицо | Местоимение | Значение |
|------|-------------|---------|
| 1 ед. | μου | мой / моя / моё / мои |
| 2 ед. | σου | твой / твоя / твоё / твои |
| 3 ед. м. | του | его |
| 3 ед. ж. | της | её |
| 1 мн. | μας | наш / наша / наше / наши |
| 2 мн. | σας | ваш / ваша / ваше / ваши |
| 3 мн. м. | τους | их (мужской род) |
| 3 мн. ж. | τις | их (женский род) |

• Το σπίτι μου. — Мой дом.
• Η φίλη μου. — Моя подруга.
• Ο φίλος σου. — Твой друг.
• Η οικογένεια σας. — Ваша семья.
• Τα δώρα τους. — Их подарки.
• Μένω με την φίλη μου. — Живу со своей подругой.
• Πώς είναι η οικογένεια σου; — Как твоя семья?

# σου
## твой, твоя, твоё, твои
### su
Употребляется после существительного с артиклем.
• Το όνομά σου. — Твоё имя.
• Τα γενέθλιά σου. — Твой день рождения.
• Η μέρα σου. — Твой день.

# του
## его
### tu
• Το σπίτι του. — Его дом.
• Ο φίλος του. — Его друг.

# της
## её
### tis
• Το σπίτι της. — Её дом.
• Η δουλειά της. — Её работа.

# μας
## наш, наша, наше, наши
### mas
• Η κουζίνα μας. — Наша кухня.
• Οι φίλοι μας. — Наши друзья.

# σας
## ваш, ваша, ваше, ваши
### sas
• Η υπηρεσία σας. — Ваша услуга.
• Το όνομά σας. — Ваше имя.

# τους
## их (мужской/смешанный)
### tus
• Οι φίλοι τους. — Их друзья.
• Το σπίτι τους. — Их дом.

# τις
## их (женский)
### tis
• Οι φίλες τις. — Их подруги (ж.р.)
