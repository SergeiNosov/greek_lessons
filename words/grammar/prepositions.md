# σε
## в, на, к, по (предлог места/направления)
### se
Предлог σε сливается с определённым артиклем:
| σε + | → |
|------|---|
| τον (муж. вин.) | στον |
| την (жен. вин.) | στην |
| το (ср. вин.) | στο |
| τους (мн. муж.) | στους |
| τις (мн. жен.) | στις |
| τα (мн. ср.) | στα |

• Σε + τον = στον
• Σε + την = στην
• Σε + το = στο

• Πάω στο σπίτι. — Иду домой. (σε + το = στο)
• Μένω στην Κύπρο. — Живу на Кипре. (σε + την = στην)
• Δίνω φαγητό στο παιδί. — Даю еду ребёнку.
• Γιορτάζω στον καφέ. — Праздную в кафе.
• Μιλάω στα ελληνικά. — Говорю по-гречески. (σε + τα = στα)

# από
## из, от, с
### apó
• Είμαι από τη Ρωσία. — Я из России.
• Από πού είσαι; — Откуда ты?
• Αυτό είναι δώρο από τον φίλο μου. — Это подарок от моего друга.
