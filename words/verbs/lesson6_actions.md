# είμαι
## я есть; быть
### íme
- είμαι, είσαι, είναι, είμαστε, είστε, είναι
- ήμουν, ήσουν, ήταν, ήμασταν, ήσασταν, ήταν
- θα είμαι, θα είσαι, θα είναι, θα είμαστε, θα είστε, θα είναι
Очень частый глагол-связка: «я есть», «он/она/оно есть», «находиться».
• Το σπίτι είναι μεγάλο. — Дом большой.
• Τα παιδιά είναι στο πάρκο. — Дети в парке.

# πηγαίνω
## иду; еду
### piyéno
Более полная форма глагола `πάω`.
- πηγαίνω, πηγαίνεις, πηγαίνει, πηγαίνουμε, πηγαίνετε, πηγαίνουν
- πήγα, πήγες, πήγε, πήγαμε, πήγατε, πήγαν
- θα πάω, θα πας, θα πάει, θα πάμε, θα πάτε, θα πάνε
• Πηγαίνω στο σχολείο. — Я иду в школу.
• Εγώ πηγαίνω στη θάλασσα. — Я иду к морю.

- ήξερα, ήξερες, ήξερε, ξέραμε, ξέρατε, ήξεραν
- θα ξέρω, θα ξέρεις, θα ξέρει, θα ξέρουμε, θα ξέρετε, θα ξέρουν

• Ο παππούς ξέρει μια καλή ιστορία. — Дедушка знает одну хорошую историю.
