# διαβάζω
## я читаю
### diavázo
Связка: `διαβάζω → διάβασα → θα διαβάσω`
Это форма настоящего времени: процесс, привычка или повтор.
• Διαβάζω κάθε βράδυ. — Я читаю каждый вечер.

# διάβασα
## я прочитал(а)
### diávasa
Связка: `διαβάζω → διάβασα → θα διαβάσω`
Это аорист: одно завершённое действие в прошлом.
• Διάβασα το βιβλίο χτες. — Я прочитал(а) книгу вчера.

# θα διαβάσω
## я прочитаю
### tha diaváso
Связка: `διαβάζω → διάβασα → θα διαβάσω`
Это простое будущее: `θα` + завершённая основа.
• Θα διαβάσω το γράμμα αύριο. — Я прочитаю письмо завтра.

# γράφω
## я пишу
### gráfo
Связка: `γράφω → έγραψα → θα γράψω`
В настоящем используется одна основа, а в аористе/будущем другая: `γραφ- / γραψ-`.
• Γράφω ένα μήνυμα τώρα. — Я пишу сообщение сейчас.

# έγραψα
## я написал(а)
### égrapsa
Связка: `γράφω → έγραψα → θα γράψω`
Начальное `ε-` помогает увидеть прошедшее время.
• Έγραψα ένα γράμμα στη φίλη μου. — Я написал(а) письмо подруге.

# θα γράψω
## я напишу
### tha grápso
Связка: `γράφω → έγραψα → θα γράψω`
Будущее берёт ту же завершённую основу, что и аорист.
• Θα γράψω μετά τη δουλειά. — Я напишу после работы.

# μαγειρεύω

• Μαγειρεύω κάθε μέρα στο σπίτι. — Я готовлю каждый день дома.

# μαγείρεψα
## я приготовил(а)
### mayírepsa
Связка: `μαγειρεύω → μαγείρεψα → θα μαγειρέψω`
Аорист показывает одно завершённое приготовление.
• Μαγείρεψα σούπα το βράδυ. — Я приготовил(а) суп вечером.

# θα μαγειρέψω
## я приготовлю
### tha mayirépso
Связка: `μαγειρεύω → μαγείρεψα → θα μαγειρέψω`
Простое будущее: единичное будущее действие.
• Θα μαγειρέψω αύριο για όλους. — Я приготовлю завтра для всех.

• Αγάπησα πολύ αυτό το μέρος. — Я очень полюбил(а) это место.

# θα αγαπήσω

Будущее использует ту же основу, что и аорист.

# μένω

# θα μείνω

Будущее повторяет аористную основу без прошедшего маркера.
• Θα μείνω εδώ απόψε. — Я останусь здесь сегодня вечером.

# θα δω

# θα πάω
