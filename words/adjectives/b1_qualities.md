# υγιεινός(-ή,-ό)
## здоровый; полезный для здоровья
### iyinós(-í,-ó)
**Муж.:** υγιεινός / υγιεινοί
**Жен.:** υγιεινή / υγιεινές
**Ср.:** υγιεινό / υγιεινά

• Η υγιεινή διατροφή είναι σημαντική. — Здоровое питание важно.
• Προσπαθώ να τρώω υγιεινά. — Стараюсь питаться здорово.
